## Setup

1. Clone GitHub Repository
2. Install requirements in the txt file
2. Rename .env.example.py file as .env
   3. Retrieve your own Gemini API key for access to the 2.5 flash model and paste your key where it says 'your_gemini_key_here' 
4. To launch the app, run the command 'streamlit run app.py' in your terminal from the root of the repository
